# Autism Support Companion

This is a mobile application designed to support individuals on the autism spectrum. The app provides features for user registration, profile creation, personalized icon creation, and recommendations.

## Features

- User registration and profile creation
- Personalized icon creation
- Personalized recommendations for lifestyle modifications and exercise routines
